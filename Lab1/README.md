# Отчёт по лабораторной работе №1
## Тема: Алгоритмы вычисления и умножения полиномов. Метод Карацубы.

### Реализованный функционал

В ходе работы были реализованы следующие алгоритмы:

*   **Вычисление значения полинома P(x):**
    *   Прямой метод через возведение в степень: сложность O(n log n)
    *   Схема Горнера: итеративный метод, сложность O(n)
*   **Умножение полиномов:**
    *   Классический метод в столбик: сложность O(n^2)
    *   Алгоритм Карацубы: рекурсивный метод, теоретическая сложность O(n^1.55).
*   **Умножение целых неотрицательных чисел:**
    *   Реализовано через представление чисел как массивов коэффициентов полинома (основание x=10). Применяются алгоритмы умножения полиномов с последующей обработкой переносов разрядов

### Методика тестирования

*   **Функциональные тесты:** Сравнение результатов разных алгоритмов на идентичных входных данных для подтверждения корректности
*   **Граничные условия:** Тестирование полиномов разной длины, обработка нулевых коэффициентов и пустых входных данных
*   **Производительность:** Автоматизированные замеры времени выполнения (использование time.perf_counter) на случайных данных больших объемов (до 10^6 элементов для вычисления и до 4096 для умножения)

### Выводы

#### Сравнение методов умножения
Алгоритм Карацубы демонстрирует преимущество над классическим методом только на больших объемах данных (N > 2048)
*   **Причины:** Высокая константа алгоритма Карацубы в Python обусловлена накладными расходами на рекурсию, создание объектов и срезы списков
*   **Оптимизация:** Для повышения эффективности рекомендуется использовать гибридный подход — переход на классическое умножение при малых длинах подзадач (N <= 64)

#### Сравнение методов вычисления значения
Схема Горнера является наиболее эффективным методом вычисления значения полинома.
*   **Преимущество:** Исключение операции возведения в степень снижает общее количество математических операций и обеспечивает строго линейную сложность O(n)
*   **Стабильность:** На экстремально больших степенях (10^6) Горнер показывает двукратное ускорение относительно прямого метода

#### Умножение целых чисел
Метод сведения умножения чисел к умножению полиномов позволяет корректно обрабатывать длинную арифметику произвольной разрядности, обеспечивая корректность за счет алгоритмов переноса разрядов
